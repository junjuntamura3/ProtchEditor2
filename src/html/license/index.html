PROTCH [フ゜ロッチ]に関して

PROTCH [フ゜ロッチ]は山崎教育システム株式会社の登録商標て゛す。
フ゜ロッチ・エテ゛ィターは、米国マサチューセッツ工科大学メテ゛ィアラホ゛か゛開発したオーフ゜ンソースソフトウェア「SCRATCH・スクラッチ」を カスタム化したフ゛ロックフ゜ロク゛ラミンク゛学習ソフトて゛す。
Protch Editor is based on Scratch from the MIT Media Lab.
